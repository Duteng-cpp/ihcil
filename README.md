# IHCIL@OUC - 海洋智能计算实验室

海洋大学智能计算实验室（Intelligent Computing for Oceanography Laboratory）官方网站。

## 关于我们

IHCIL 是海洋大学计算机科学与技术学院的研究实验室，专注于人工智能在海洋科学中的应用。

## 研究方向

- 海洋预报与预测
- 雷达数据反演
- 气象数据分析
- 人工智能海洋学

## 网站构建

本网站使用 [Hugo](https://gohugo.io/) 静态网站生成器和 [Wowchemy](https://wowchemy.com/) 主题构建。

### 本地开发

```bash
# 安装 Hugo (如果还没有安装)
# Windows: choco install hugo
# 或从 https://gohugo.io/installation/ 下载

# 启动开发服务器
hugo server

# 访问 http://localhost:1313
```

### 构建网站

```bash
hugo
```

生成的静态文件在 `public/` 目录中。
